Grammarnazi

A module for Willie the IRC bot written in python. It warns the user who uses SMS lingo during an IRC session.

Fun with Willie and grammarnazi

11 September 2014

Last summer I saw someone using an IRC bot and I got curious on how to use it. I tried searching online but I thought I wouldn't understand. Recently when I was exploring python, I felt like giving it a try. I came across Willie the IRC bot. During our IRC sessions on #fsmk channel @sant0sh always warns someone who uses SMS lingo. That's how I got this idea of writing this module for willie. I haven't completed the code yet. I want to make it more efficient. Here's how you can run/test it: 

$ sudo apt-get install python-pip
$ sudo pip install willie
$ willie

That's how you can run Willie. BTW you should have python 2.7 installed. When you run for the first time, it asks for configuration settings. Keep hitting Enter for options with [square brackets] . Fill the other details.
If you want to change the configuration manually. 

$ cd .willie
$ vi default.cfg

To test my module, just download it and paste the grammarnazi.py file in the modules folder inside .willie/. When you run willie the next time, the module is loaded automatically.
